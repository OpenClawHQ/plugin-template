# Changelog

All notable changes to this project will be documented in this file.

Format based on [Keep a Changelog](https://keepachangelog.com/).

## [0.1.0] - 2026-03-04

### Added

- Initial plugin template with `PluginBase` and `PluginConfig` classes.
- Example plugin implementation in `plugin.py`.
- Configuration schema in `config.py`.
- Test suite with pytest.
- CI workflow with Ruff linting, mypy type checking, and multi-version testing.
- Project documentation (README, LICENSE, CHANGELOG).
